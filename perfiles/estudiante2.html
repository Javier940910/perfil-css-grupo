Conclusiones sobre la especificidad de CSS

Durante el desarrollo de esta actividad comprendí que la especificidad en CSS es muy importante porque determina qué estilo se aplica cuando existen varias reglas que afectan al mismo elemento. Entender cómo funcionan los selectores permite tener mayor control sobre el diseño de la página y evitar conflictos entre estilos. Además, aprendí que usar selectores simples y bien organizados ayuda a mantener el código más limpio y fácil de mantener.

Selectores avanzados que me costó entender

  • :hover – Al principio me costó entender que este selector se activa cuando el usuario pasa el cursor sobre un elemento.
  • :first-child – Fue difícil entender que selecciona únicamente el primer elemento dentro de un contenedor.
  • :nth-child() – Este selector permite seleccionar elementos según su posición dentro de una lista, lo cual requiere entender bien cómo funciona la estructura del HTML.

Metodología Mobile-First

La metodología Mobile-First consiste en diseñar primero las interfaces pensando en dispositivos móviles y posteriormente adaptarlas a pantallas más grandes como tablets y computadores. Este enfoque es importante porque actualmente muchos usuarios acceden a los sitios web desde sus teléfonos. En el desarrollo de este perfil, la idea principal fue construir primero una estructura sencilla y clara que funcione correctamente en pantallas pequeñas. Luego, a partir de esa base, se pueden agregar mejoras visuales o ajustes para pantallas más grandes. Esto permite que la página sea más accesible, rápida y fácil de usar.
